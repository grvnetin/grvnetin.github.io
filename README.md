# grvnetin.github.io

This is the repository for the personal website of grvnetin.

## Description

This repository contains the source code and content for the personal website hosted at [grvnetin.github.io](https://grvnetin.github.io).

## Structure

- `index.html`: The main landing page of the website.
- `css/`: Directory containing CSS files for styling the website.
- `js/`: Directory containing JavaScript files for website functionality.
- `images/`: Directory containing images used on the website.

## Usage

To view the website locally, open `index.html` in your web browser.
